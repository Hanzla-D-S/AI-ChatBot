# ShopEasy AI Customer Support Chatbot

An AI-powered customer support chatbot built with RAG, LangChain, FastAPI, and Streamlit.

## Tech Stack
- LangChain + OpenAI GPT-3.5
- FAISS Vector Store
- FastAPI Backend
- Streamlit Frontend
- Docker
